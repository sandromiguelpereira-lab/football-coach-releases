# ⚽ Football Coach Platform

Plataforma completa de gestão para treinadores de futebol.

## 🎯 Funcionalidades Atuais (Sprint 1)

### ✅ Implementado
- Sistema de autenticação (Login/Registo)
- Dashboard principal
- Navegação com menu lateral
- Design responsivo (mobile e desktop)
- Integração com Firebase

### 🚧 Próximo (Sprint 2)
- Gestão completa de atletas
- Fichas individuais
- Estatísticas e avaliações

## 🛠️ Tecnologias

- **Frontend**: React 18 + Vite
- **UI Framework**: Material-UI (MUI) 5
- **Backend**: Firebase (Authentication + Firestore + Storage)
- **Routing**: React Router v6
- **Idioma**: Português de Portugal

## 📦 Comandos

```bash
# Instalar dependências
npm install

# Desenvolvimento
npm run dev

# Build produção
npm run build

# Preview do build
npm run preview
```
